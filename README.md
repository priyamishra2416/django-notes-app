
# 📝 Django Notes App

A simple and responsive Notes App built using Django.  
Users can add and manage notes with a clean UI and smooth experience.

---

## 🚀 Features

- Add Notes
- Responsive Design
- Clean User Interface
- Django Backend Integration
- SQLite Database

---

## 🛠 Tech Stack

- Python
- Django
- HTML
- CSS
- Bootstrap
- SQLite

---

## ⚙️ Installation

```bash
git clone https://github.com/priyamishra2416/django-notes-app.git

cd django-notes-app

pip install -r requirements.txt

python manage.py runserver
